# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

A sandbox project for testing and documenting [Podman](https://podman.io/) with a goal to create a markdown-based presentation using [reveal.js](https://revealjs.com/).

## Documentation Structure

- `docs/planning/` - Project planning and goals
- `docs/research/` - Research notes and component documentation
- `docs/diagrams/` - Excalidraw architecture diagrams
- `docs/assets/logos/` - Logo images for presentations (max 150px wide)

## Key Resources

### Architecture Diagrams (Excalidraw)
- `docs/diagrams/docker-architecture.excalidraw` - Docker architecture (daemon-based)
- `docs/diagrams/podman-architecture.excalidraw` - Podman architecture (daemonless)

### Research Documents
- `docs/research/docker-architecture-components.md` - Docker component documentation
- `docs/research/podman-architecture-components.md` - Podman component documentation
- `docs/research/docker-podman-commands.md` - Docker/Podman command reference
- `docs/research/docker-registry-components.md` - Registry (Images, Extensions, Plugins)

## Presentation

Using [reveal.js](https://revealjs.com/) with [markdown](https://revealjs.com/markdown/) option for slides.

## Code Style

- 2 spaces for indentation
- UTF-8 encoding
- LF line endings
- Max line length: 120 characters
